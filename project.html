neural network classifier

🧠 Interactive demo would go here (MNIST digit classifier)

This project was my deep dive into understanding neural networks from the ground up. Instead of using high-level libraries, I implemented backpropagation, gradient descent, and activation functions manually to truly grasp how these systems learn.

The classifier achieves 94% accuracy on the MNIST handwritten digit dataset. What I loved most about this project was the moment when the network finally "clicked" and started recognizing patterns - there's something magical about watching a machine learn to see.

This project taught me patience, debugging skills, and gave me a solid foundation in machine learning that I continue to build upon in my other work.

community data visualizer

🌐 Live interactive charts and community engagement metrics

Partnering with local community organizations, I created an interactive dashboard that transforms raw engagement data into beautiful, understandable visualizations.

The tool helps organizations track volunteer participation, event attendance, and community impact over time. What makes this special is how it bridges the gap between complex data and actionable insights for non-technical users.

The interface prioritizes accessibility and ease of use, with color-blind friendly palettes and screen reader compatibility built in from day one.

ecological data insights

🌱 Biodiversity visualizations and statistical analysis

This research project analyzed biodiversity patterns in urban environments, examining how city planning affects local ecosystems and wildlife populations.

Using R and statistical modeling, I processed large datasets of species observations, weather patterns, and urban development metrics to identify correlations and trends. The resulting visualizations helped communicate complex ecological relationships to both academic and public audiences.

What I loved about this project was how it combined rigorous data science with environmental storytelling, showing how numbers can reveal the hidden connections in our natural world.
